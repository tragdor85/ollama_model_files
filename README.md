# Ollama Modelfiles

Modelfiles optimized for running on **MacBook Pro M1 Max (32GB)** using [Ollama](https://ollama.com).

Each modelfile in this repo includes tuned parameters for:

- **Performance** — optimized `num_gpu`, `num_thread`, `num_ctx` for M1 Max architecture
- **Coding** — sampling parameters tuned for code generation tasks
- **Stability** — settings to prevent mid-output cutoff and ensure consistent responses
